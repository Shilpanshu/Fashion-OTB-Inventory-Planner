### **Overview**
A Streamlit-based Merchandising Dashboard that automates Open-to-Buy (OTB) planning and Inventory Analysis. Built to replace manual VLOOKUP workflows with Python-based logic.

### **Key Features**
* **Trinity KPI Deck:** Calculates Gross Margin %, Weeks of Cover (WOC), and Sell-Through % instantly.
* **Inventory Health Analysis:** Identifies "Slow-Moving" vs. "Low-Margin" inventory risks using a Scatter Plot Matrix.
* **Automated OTB Calculator:** Dynamic "What-If" modeling to calculate Buy Budgets based on Target Growth & Cover.
* **Fashion Logic:** Simulates real-world retail constraints (Size Curves, Seasonality, and Markdown Status).

### **How to Use**
1.  Upload your Sales Data CSV (or use the built-in Demo Data).
2.  Filter by **Category** (Menswear/Womenswear) or **Season** (SS/AW).
3.  Use the **OTB Slider** to adjust your "Planned Growth" and see the budget update in real-time.

### **Tech Stack**
* **Python:** Core Logic
* **Pandas:** Data Processing & Merging
* **Streamlit:** Web Application Framework
* **Plotly:** Interactive Visualizations
